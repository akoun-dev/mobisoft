# CONTEXTE TECHNIQUE - AFRICAHUB

## 1. FRONTEND
### Frameworks
- React 18+ avec Vite
- Tailwind CSS + Headless UI
- React Router v6
- React Hook Form + Zod (validation)

### State Management
- Zustand (state global)
- React Query (data fetching)

### Autres
- Framer Motion (animations)
- Recharts + D3.js (visualisation)
- Service Workers (PWA)

## 2. BACKEND
### Core
- Node.js 18+
- Express.js / Fastify
- Supabase (auth/storage)

### Outils
- Prisma (ORM)
- Joi/Yup (validation)
- Swagger/OpenAPI (docs)
- Jest + Supertest (tests)

## 3. BASE DE DONNÉES
- PostgreSQL (Supabase)
- Redis (cache)
- Elasticsearch (recherche)
- ClickHouse (analytics)

## 4. INFRASTRUCTURE
### Hébergement
- Vercel (Frontend)
- Railway/Fly.io (Backend)
- Cloudflare (CDN)

### Monitoring
- Sentry (erreurs)
- LogRocket (session replay)
- Plausible/Matomo (analytics)

### CI/CD
- GitHub Actions
- Tests automatisés
- Déploiements progressifs

## 5. SÉCURITÉ
- Supabase Auth + JWT
- 2FA (TOTP)
- TLS 1.3
- WAF (Cloudflare)
- Rate limiting
